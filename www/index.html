About WebJar

Runnable

A full-stack, executable Java web application that ships in a jar file. It takes one command to fire up a dynamic website that serves content via a REST interface.

Learn More

Versatile

Besides serving web content, the application is great for prototyping because you can serve different templates to view the same content

Extensible

In addition to using the built-in functionality, you can easily extend the framework to handle your own unique requirements.

Learn More

The rapid evolution of computer hardware platforms coupled with that of software platforms and services has made it incredibly convenient and practical to bring the power of a battle-tested development platform like Java to the fingertips of both the designer and the developer. You can now easily lease a virtual private server with enormous bandwidth and memory from a vendor like Vultr for less than $5 a month, which is phenomenal. There is no limit to what you can begin doing with such a set up. You get to choose you own software stack to run and choose how to run it.

With webjar (the framework), you get to run your purely HTML5/JavaScript application in a solidly stable Jetty web server and serve content with a REST-based api powered by Jersey, Java's own reference implementation of the REST API specification. You have the versatility of Spring for inversion of control to decouple components in your application, and a backend for both relational data (sqlite) and a nosql, document-based data (couchbase lite). This affords your application database needs the flexibility to evolve unabated.

Javascript has also matured into a beautiful and formidable development platform with numerous frameworks to show for this in the view layer. Employing asynchronous techniques like AJAX makes using traditional methods of rendering HTML on the server side unsuitable for the responsiveness expectations of the web in the present day. Webjar respects this wisdom and conventionally leaves the choice of view components in the hands of the user.

WebJar Features

The webjar framework is primarily powered by:

  • Jetty web server
  • Jersey for REST api
  • Spring DI/Jackson json
  • Sqlite/Couchbase lite
  • VueJs and JQuery
  • Bootstrap with Font Awesome
