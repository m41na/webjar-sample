REST interface

The application content is served by a stateless REST interface which can be consumed by any authorized client.

Resource Access

Access to application content is passed through headers is controlled to allow access based on roles (admin or guest).

Testable Endpoints

The admin section provides a great utility for testing your REST endpoints. This is one indispensable tool for a developer
